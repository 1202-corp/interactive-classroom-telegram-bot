# Interactive Classroom Telegram Bot

Telegram bot for the Interactive Classroom Platform - a web application for conducting interactive classes in university auditoriums.

## Overview

Telegram bot for integrating the platform with course group chats. Simplifies sending session invitations and provides quick access to interactive classes.

## Key Features

- Sending links and QR codes for session connection
- Commands for teachers (starting sessions, getting information)
- Integration with course group chats
- Quick access to active sessions
