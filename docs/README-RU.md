# Документация Telegram-бота

Telegram-бот для платформы интерактивных занятий Interactive Classroom Platform.

## Описание

Telegram-бот для интеграции платформы с групповыми чатами курсов. Упрощает рассылку приглашений на сессии и предоставляет быстрый доступ к интерактивным занятиям.

## Основные функции

- Рассылка ссылок и QR-кодов для подключения к сессии
- Команды для преподавателей (запуск сессий, получение информации)
- Интеграция с групповыми чатами курсов
- Быстрый доступ к активным сессиям

## Технологии

- Telegram Bot API
- Интеграция с Backend API платформы
- Команды для управления сессиями
